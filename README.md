## Overview

The goal of this project was to build a full admin dashboard layout using
CSS Grid as the primary layout tool.

## What I Learned

- Creating complex layouts using CSS Grid
- Nesting grids within grids
- Structuring semantic HTML for dashboards
- Managing spacing and alignment without Flexbox-heavy layouts
- Building reusable UI sections (cards, panels, navigation)

## Features

- Sidebar navigation
- Header with search, user info, and actions
- Projects grid section
- Announcements and trending panels

## Notes

- This project is not responsive, as responsive design is covered later.
- Icons and images are placeholders.
- No JavaScript is used.

## Tools Used

- HTML5
- CSS Grid
- Google Fonts (Roboto)
